# Daily Trending Info

## Overview
Daily-Trending-Info collates and processes real-time trending data from various platforms. The app delivers curated trends for different categories such as news, entertainment, technology, and more. This project emphasizes user-content curation coupled with analytics.

---

## Features

### 1. Data Collection
The system utilizes multiple APIs to collect categorized trending data seamlessly. Also, real-time API fallbacks are implemented to ensure continuity in case a primary API call fails. For reference on backup API implementations, see the [Advanced APICallScript.py](scripts/ApiCallBackupScript.py).

### 2. User Customizations
User-specific curated preferences available in-app for trend filtering based upon metaheuristic algorithms.
